This is a project to help automate planning for businesses using AI.


How to use:
1. Download the **environment.yaml** file as a virtual environment
2. Create a .env file by following the scaffold in **.env.example**
3. Make and add your OLLAMA or any other LLM API Key inside the **.env** file
4. Add your Google **credentials.json** inside _credentials/_ directory
5. While still in the same respository, run the command: `streamlit run app.py`
6. Test the project out! :)
